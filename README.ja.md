# chrome-boxnotes

Box Notes の本文を Markdown に変換して `.md` ファイルとしてダウンロードする Chrome 拡張 (Manifest V3) です。

## 機能

- Box Notes エディターのメニューにダウンロードボタンを追加します
- 現在表示中のノートを Markdown に変換し、`<ノートタイトル>.md` として保存します
- 対応要素:
  - 見出し (H1〜H3)
  - 段落
  - 太字・イタリック・取り消し線
  - インラインコード・コードブロック (言語 info string 対応。`sh`, `cpp`, `csharp` など)
  - リンク
  - 箇条書き・番号付きリスト (ネスト対応)
  - チェックリスト (`[ ]` / `[x]`)
  - ブロック引用
  - 区切り線
  - GFM 形式のテーブル (`colspan` / `rowspan` 対応)

## インストール

この拡張は Chrome ウェブストアには公開していません。パッケージ化されていない拡張機能として読み込んでください。

1. リポジトリを clone してビルドします:
   ```bash
   npm install
   npm run compile
   ```
2. Chrome で `chrome://extensions` を開きます。
3. 右上の **デベロッパーモード** を有効にします。
4. **パッケージ化されていない拡張機能を読み込む** をクリックし、`dist/` ディレクトリを選択します。

## 使い方

1. Box Notes のドキュメントを開きます (`https://notes.services.box.com/p/note?...`)。
2. エディター上部のメニューにダウンロードボタンが表示されます。
3. クリックすると `<ノートタイトル>.md` としてダウンロードされます。

## 制約

- 現在の Box Notes の DOM 構造に依存しています。Box 側の変更により抽出が動作しなくなる可能性があります。
- ノートに埋め込まれた画像はエクスポートされません。
